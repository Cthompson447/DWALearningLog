Learning Outcomes - Practical 0

Learning outcomes:

  • Practical 0 involved replicating an example webpage "WorldBooks" this involved familiarising myself again with various html tags to build this form.
  • Creation of this form involved the following HTML constructs.
    • Images
    • Horizontal Rule
    • Headers h2 h3
    • Paragraph tags
    • Text boxes
    • Tables
    • Radio Buttons
    • Checkboxes
    • Buttons
  • In order to display the unicode currency symbol for pounds correctly I had to use & before the word "pound" to display this symbol correctly.
  • I also experimented with wrapping multiple elements within 1 tag in order to get them to display on the page on a single line - see radio buttons.
