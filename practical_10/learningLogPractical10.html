    • Task 2 & 3
      • Task 2 proved to be a bit trickier than task 1, I did some research and came across some examples on W3 schools for how to implement this. The slideshow works by running document.getElementsByClassName on the images which all have the same class to be part of the slideshow. Once we get the HTMl collection of these images, we set all display style properties to none. One image has it's style display property assigned to block. The script then continues to run but with an index change of +1 on the images in the collection. This allows it to loop through continuously displaying the next image each time.
      • I implemented Task 3 while working on 2, it was a small code change to add a timeout to the function to force it to update every 3 seconds. When I tested this on the site it looked good, all 5 images loop indefinitely when on the webpage.
