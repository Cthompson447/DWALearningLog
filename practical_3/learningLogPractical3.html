  • Task 4:

    Task 4 involves finding and explaining 6 errors in the given in the practical document.

    1. The opening script tag wasn't closed before javascript code - it would mean the javascript would be treated as tag attributes and would do nothing.
    2. 'var 1function="learning" ;' has 2 problems, you cannot start a variable with a number and function is a reserved keyword and cannot be used for variable assignment.
    3. 'var course' is not referenced because of a spelling mistake in 'document.write("I am " + learning +" "+ corse + " for "+ year +" years,"+ type);'
    4. bgcolour does not work because the style attribute uses the american spelling 'color'
    5. 'var type="full time"' and ' year +=2' don't have semicolons at end of line, not best practice
    6. 'text' is a deprecated attribute, 'color' should be used instead for text
    7. 'var course=imd/ict;' is incorrect - you cannot divide a string by a string. Assuming this is imd or ict, this variable should be in quotes e.g. var course="imd/ict".
