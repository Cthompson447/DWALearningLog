  • Task 1
    • Task 1 Part 3
      • This task involves describing what the phone validation portion of the provided script does:
      • if (!document.userSurvey.phone.value) - this mean if no value is present
      • window.alert("Phone number missing. Please enter a valid phone number to continue."); - alert fired in browser to notify user number is missing
      • document.userSurvey.phone.focus(); - focus the phone input element
      • return false; - does what it says on the tin, return false value from validation script
      • else clause - this means we have a phone number, continue with further validation
      • var numbersOnly = ""; var chars = ""; var phoneNo = document.userSurvey.phone.value; - initialise 3 variables, 2 empty and one with value of what was entered into our phone input box
      • for (i = 0; i < phoneNo.length; i++) - loop to iterate length of number
      • chars = phoneNo.substring(i,i+1); - get next character in position of phone number string
      • if (chars >= "0" && chars <= "9") { numbersOnly = numbersOnly + chars; } - if our character is greater than or equal to 0 and less than or equal to 9, we add it to our numbers only list.
      • if (numbersOnly.length != 13) { window.alert("Incorrect phone number format.You must enter 13 numbers."); document.userSurvey.focus(); return false; } - numbers have 13 characters, if not equal to length of 13, alert of invalid input, focus input box and fail validation
      • else { var areacode = numbersOnly.substring(0,2); var leading0 = numbersOnly.substring(2,3); var exchange = numbersOnly.substring(3,5); var ext1 = numbersOnly.substring(5,9); var ext2 = numbersOnly.substring(9); - declaration of substrings to specify number positions
      • var newNumber =( "+" + areacode + " " +"(" + leading0 + ")" + exchange + " " + ext1 + "-" + ext2); document.userSurvey.phone.value = newNumber; return true; } - formatting of phone number using above substring values to dictate when to add symbols, once this has been done validation returns true.


    • Task 1 Part 5
      • This task involves describing what the email validation of the provided script does:
      • if (!document.userSurvey.eMail.value) { window.alert("E-mail Address missing. Please enter a valid E-mail address to continue."); document.userSurvey.eMail.focus(); return false; - If no email is present, alert the user, focus email input and return false to fail validation
      • var emailAddress = document.userSurvey.eMail.value; var atLoc = emailAddress.indexOf("@",1); var dotLoc = emailAddress.indexOf(".",atLoc+2); var len = emailAddress.length; - Declare 4 variables, emailAddress is value of input box, atLoc specifies where to find @ e.g search from index of 1 onwards for @ symbol. dotloc is 2 characters after @ symbol declaration and length is size of our email string.
      • if (atLoc > 0 && dotLoc > 0 && len > dotLoc+2) { return true; } - If we meet all of our rules as specified in our variables above, return true meaning validation successful.
      • else { alert("Invalid E-mail address! Please enter your email address again."); document.userSurvey.eMail.focus(); return false; } - If we have an email address entered but fail to meet our validation rules, we flag this as incorrect, alert user, focus email input and return false to fail our validation.
